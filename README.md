# Document-Database-Project

- Suggesting a usage scenario for MongoDB store with at least three different collections (system for e-commerce websites).
- Proposing 4 data queries (statements retrieving data) appropriate for the envisioned usage scenario.
- Implementing queries.
- Adding +- 20 documents to the database.
- Setting up the database, so it has two shards each with a master and two ordinary replicas and one hidden and unvoting replica.
